# Quickstart: Premium Progression Map

Since this is purely a UI layer update, there are no new endpoints, CLI commands, or APIs to interact with.

## How to Test the Progression Map

1. Launch the app and navigate to the **Progression Map** (e.g., via the Spatial Span preview card).
2. Observe that **all 3 vertical tracks** are visible.
3. Observe that only nodes the player has unlocked are fully illuminated, while the rest are dimmed.
4. Observe the premium rendering style:
   - Lines connecting the nodes now have gradient strokes and subtle glows instead of flat colors.
   - Nodes feature dimensional rendering (e.g., inner gradients, outer blurs) matching their respective track colors (Blue, Pink, Lime).
